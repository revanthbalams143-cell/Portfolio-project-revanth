# Portfolio Website 🚀

A modern and fully responsive personal portfolio website built to showcase my skills, projects, and passion for web development. This project reflects my ability to create clean UI designs, responsive layouts, and interactive user experiences using core frontend technologies.

---

## 📌 Project Overview

This portfolio website was designed and developed to represent my professional profile as a Full Stack Developer.
The main goal of this project is to create a strong digital presence that highlights:

* My technical skills
* Featured projects
* Contact information
* Responsive UI/UX design abilities
* JavaScript-based interactivity

The website focuses on smooth navigation, modern animations, mobile responsiveness, and clean code structure.

## ⚡ Key Functionalities

### Dynamic Section Rendering

Implemented JavaScript functions to dynamically show and hide sections for a smoother single-page application experience.

### Scroll Animations

Used the Intersection Observer API to trigger fade-in animations when elements appear on screen.

### Responsive Navigation

Built a fully responsive navigation system with hamburger menu functionality for smaller devices.

### Modern Visual Design

Designed a clean and professional interface using gradients, animations, hover effects, and modern UI principles.
